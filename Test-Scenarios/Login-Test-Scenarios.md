# Login Test Scenarios

## Scenario 1: Verify login with valid credentials
**Expected Result:**  
User should be logged in successfully.

---

## Scenario 2: Verify login with invalid password
**Expected Result:**  
System should display an appropriate error message.

---

## Scenario 3: Verify login with empty fields
**Expected Result:**  
System should show validation message for mandatory fields.

---

## Scenario 4: Verify login with invalid email format
**Expected Result:**  
System should display email format validation message.

---

## Scenario 5: Verify password field security
**Expected Result:**  
Password should be masked and not visible in plain text.

---

## Scenario 6: Verify login functionality after multiple failed attempts
**Expected Result:**  
System should handle multiple failed login attempts as per security requirements.
