# TourMyIndia (TMI)

## Project Overview

TourMyIndia (TMI) is a travel and tourism management platform that allows administrators to manage 
travel destinations, tour packages, hotels, blogs, and travel-related content. 
The platform consists of a Website and an Admin Panel. The website enables users to explore destinations 
and submit travel enquiries through a lead form, while the Admin Panel is used to manage website content, leads, and system activities.
