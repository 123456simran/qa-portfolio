# Tasty Punjab

## Project Overview

Tasty Punjab is an online food ordering platform that enables users to browse restaurants, 
explore menus, place food orders, and receive invoices after successful orders. 
The project consists of a customer website, a mobile application, and an admin panel for managing the complete food ordering process.

## Domain

Food Ordering

## Platform

- Website
- Mobile Application
- Admin Panel

## Project Type

Food Ordering & Restaurant Management System

## Key Features

- User Registration & Login
- Restaurant Management
- Menu & Item Management
- Food Categories
- Search & Filters
- Cart Management
- Online Food Ordering
- Order Management
- Order History
- Invoice Generation
- User Profile Management
- Admin Dashboard
