# Femminex

## Project Overview

Femminex is an online jewellery shopping platform that allows users to browse, explore, 
and purchase jewellery products. The platform includes a customer website, a mobile application, 
and an admin panel for managing products, orders, users, and inventory.

## Domain

E-Commerce (Jewellery)

## Platform

- Website
- Mobile Application
- Admin Panel

## Project Type

Online Jewellery Shopping Platform

## Key Features

- User Registration & Login
- Product Categories
- Product Search & Filters
- Product Details
- Wishlist
- Shopping Cart
- Online Order Placement
- Order Tracking
- User Profile Management
- Notifications
- Admin Dashboard
- Product & Inventory Management
- Order Management
