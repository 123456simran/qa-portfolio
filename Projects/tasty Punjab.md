# Project 1 - Tasty Punjab

## Project Overview

Tasty Punjab is an online food ordering platform that enables users to browse restaurants, explore menus, 
place food orders, and generate invoices. The project includes a customer website, a mobile application, 
and an admin panel for managing restaurants, menu items, orders, and invoices.

## Domain

Food Ordering

## Platform

- Website
- Mobile Application
- Admin Panel

## Project Type

Food Ordering & Restaurant Management System

## Purpose

To provide an online platform where customers can order food while enabling administrators and restaurant 
partners to manage restaurants, menu items, customer orders, and invoices efficiently.

## Key Features

- User Registration & Login
- Restaurant Management
- Menu & Item Management
- Food Search & Categories
- Cart Management
- Order Placement
- Order Tracking
- Invoice Generation
- User Profile Management
- Admin Dashboard
