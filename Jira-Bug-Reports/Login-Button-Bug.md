# Bug Report: Login Button Not Working

## Bug ID
BUG-001

## Title
Login button does not respond when valid credentials are entered.

## Description
When the user enters valid email and password and clicks on the Login button, the user is not redirected to the dashboard.

## Environment
- Application: Web Application
- Browser: Chrome

## Steps to Reproduce
1. Open the login page.
2. Enter valid email.
3. Enter valid password.
4. Click on Login button.

## Expected Result
User should be successfully logged in and redirected to the dashboard.

## Actual Result
User remains on the login page after clicking the Login button.

## Severity
High

## Priority
High

## Status
Open
