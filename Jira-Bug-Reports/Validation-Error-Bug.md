# Bug Report: Validation Message Not User Friendly

## Bug ID
BUG-002

## Title
Technical validation message is displayed to users.

## Description
When a user enters invalid data, the application displays a technical error message instead of a user-friendly message.

## Environment
- Application: Web Application

## Steps to Reproduce
1. Open the form page.
2. Enter invalid data in the field.
3. Submit the form.
4. Observe the error message.

## Expected Result
Application should display a clear and user-friendly validation message.

Example:
"Please enter a valid email address."

## Actual Result
Technical error message is displayed.

## Severity
Medium

## Priority
Medium

## Status
Open
