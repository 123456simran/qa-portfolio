# Android Mobile Testing Checklist

## Installation Testing
- Verify the app installs successfully.
- Verify the app launches without crashes.

## UI Testing
- Verify all screens display correctly.
- Verify buttons and icons are visible.
- Verify text is readable.

## Functional Testing
- Verify login functionality.
- Verify logout functionality.
- Verify navigation between screens.

## Network Testing
- Verify app works on Wi-Fi.
- Verify app works on Mobile Data.
- Verify behavior when internet is disconnected.

## Permission Testing
- Verify Camera permission.
- Verify Storage permission.
- Verify Location permission.

## Performance Testing
- Verify app launch time.
- Verify smooth scrolling.
- Verify no unexpected crashes.
