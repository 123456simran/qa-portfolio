# QA Testing Concepts

This section contains basic concepts of software testing.

## Included Topics:

### Smoke Testing
- Verifying that the build is stable and major functionalities are working.

### Sanity Testing
- Verifying specific changes or bug fixes after minor updates.

### Functional Testing
- Testing application features according to requirements.

### Regression Testing
- Verifying that existing functionality is not impacted after changes.

### Retesting
- Testing the specific defect again after it has been fixed.

### Exploratory Testing
- Testing the application by exploring different user flows and scenarios.
